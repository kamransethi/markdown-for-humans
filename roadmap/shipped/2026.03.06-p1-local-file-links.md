# Task: Local File Picker for Link Insertion

## 1. Task Metadata
- **Task name:** Link Local File Picker
- **Slug:** link-local-file
- **Status:** shipped
- **Created:** 2026-03-06

## 2. Context & Problem
When users try to insert a link via the "File" option in the modal, they do not get a standard OS disk file picker. They need the ability to easily attach local files.

## 3. Desired Outcome & Scope
- Create an OS file picker when the user clicks "File" in the link modal.
- Once a file is picked, automatically copy it into the workspace's designated `media/` folder (or equivalent assets folder).
- Auto-fill the relative file name into the link's "text to display" and the URL field.

## 4. Work Breakdown
- Add a file input or invoke a VS Code open dialog command (depending on extension environment APIs).
- Implement file copying logic to read from OS disk and save to the workspace `media/` folder asynchronously.
- Ensure the editor inserts a relative markdown link (e.g., `[document.pdf](./media/document.pdf)`).
