## 2. Recommended Features
To elevate the markdown editor to a fully-featured, best-in-class product:

1. **Paste URL over Text:** Auto-convert selected text to a Markdown link when pasting a valid URL.

3. **Word & Reading Time Counter:** A sleek status bar metric estimating read time and exact word counts.
4. **Contextual Block Handles (Notion-style):** A `++` or `⋮⋮` handle next to text blocks to easily drag or open a quick formatting menu.
5. **Smart Links:** Drag drop other files from the file browser into the editor to create a link to it.

8. **Format Painter:** Tool to copy inline markups (bold, italic, color) and apply them rapidly via selection.
9. **Image Drag-and-Drop:** Allowing local image files to be dragged right onto the editor, auto-saving into `media/` and generating `![alt](...)` elements
10. **Collapsible Sections:** Expand/collapse behavior under H1/H2/H3 headers for easier reading logic inside the visual editor (folding).

*(These will be prioritized iteratively in subsequent sprints).*
